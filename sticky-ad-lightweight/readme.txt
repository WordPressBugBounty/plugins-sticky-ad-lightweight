Display Sticky Ad in the bottom of your screen easily

== Description ==

Start earning more with Sticky Footer Ads with our Plugin. After installing the plugin, just go to *Settings > Sticky Ads LightWeight* and input your AD Code.


== Installation ==

1. Upload `sticky-ads-lightweight` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to *Settings > Sticky Ads LightWeight* to configure.
